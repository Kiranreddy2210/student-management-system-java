# Student Management System (Java)

## 📌 Description
A console-based Java application to manage student records using Core Java concepts.

## 🚀 Features
- Add student
- Update student details
- Delete student
- Search student

## 🛠️ Technologies Used
- Java (Core Java)
- OOP Concepts
- Collections (ArrayList)
- Exception Handling

## 📷 How to Run
1. Compile: javac StudentManagementSystem.java
2. Run: java StudentManagementSystem
